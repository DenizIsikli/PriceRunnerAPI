1. Base URL
The base URL for the PriceRunnerAPI is `https://www.example.com/api/`.

2. Versioning
To ensure backward compatibility, the API is versioned. The current version is `v1`. You should specify the version in the URL when making requests.

3. Authentication
Authentication is not required for most endpoints. However, some endpoints may require API key authentication in the future.

4. Errors
The API uses standard HTTP status codes to indicate the success or failure of a request. Detailed error messages will be included in the response body.

5. Endpoints

5.1 Search for a Product
   - **URL:** `/search/<product_name>`
   - **Method:** GET
   - **Description:** Searches for a product by name and returns a list of matching products.
   - **Parameters:**
     - `product_name` (string): The name of the product to search for.
   - **Example Request:**
     ```
     GET https://www.example.com/api/search/iphone
     ```
   - **Example Response:**
     ```
     [
       {
         "name": "iPhone 13",
         "info": "Apple iPhone 13, 128GB, Blue",
         "price": 999,
         "link": "https://www.example.com/products/123"
       },
       {
         "name": "iPhone 12",
         "info": "Apple iPhone 12, 64GB, Black",
         "price": 799,
         "link": "https://www.example.com/products/124"
       }
     ]
     ```

5.2 Get a List of Products
   - **URL:** `/products`
   - **Method:** GET
   - **Description:** Retrieves a list of all products.
   - **Example Request:**
     ```
     GET https://www.example.com/api/products
     ```
   - **Example Response:**
     ```
     [

         "name": "iPhone 13",

         "price": 799,
         "link": "https://www.example.com/products/124"
     ```

5.3 Create a New Product
   - **URL:** `/products`
   - **Method:** POST
   - **Description:** Creates a new product entry.
   - **Parameters:** JSON object with product details.
   - **Example Request:**
     ```
     POST https://www.example.com/api/products
     {
       "name": "Samsung Galaxy S21",
       "info": "Samsung Galaxy S21, 128GB, Gray",
       "price": 899,
       "link": "https://www.example.com/products/125"
     }
     ```
   - **Example Response:**
     ```
     {
       "name": "Samsung Galaxy S21",
       "info": "Samsung Galaxy S21, 128GB, Gray",
       "price": 899,
       "link": "https://www.example.com/products/125"
     }
     ```

5.4 Delete a Product
   - **URL:** `/products/<product_id>`
   - **Method:** DELETE
   - **Description:** Deletes a product by its ID.
   - **Parameters:**
     - `product_id` (integer): The ID of the product to delete.
   - **Example Request:**
     ```
     DELETE https://www.example.com/api/products/125
     ```
   - **Example Response:**
     ```
     {
       "name": "Samsung Galaxy S21",
       "info": "Samsung Galaxy S21, 128GB, Gray",
       "price": 899,
       "link": "https://www.example.com/products/125"
     }
     ```
